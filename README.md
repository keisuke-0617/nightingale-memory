# ナイチンゲールメモリー

ナイチンゲールメモリーは、高齢者の皆さんが思い出を共有し、生活の中で必要となる支援を受けられるようにするためのアプリケーションです。本リポジトリはその開発を行うためのものです。

## プロジェクトの背景

日本をはじめ多くの国では高齢化が進み、家族や地域社会において高齢者のサポートが重要な課題となっています。デジタル技術を活用して、高齢者がこれまで積み重ねてきた経験や思い出を簡単に共有できる仕組みを作ることは、世代を超えた交流やコミュニティづくりにもつながります。また、日常のちょっとした困りごとをアプリを通じて周囲に相談できれば、孤立を防ぎ、より安心して暮らすことができます。

## 主な機能 (予定)

- 写真や動画、文章による思い出の記録と共有
- 身近な人や介護支援者とのチャット機能
- 日常生活に必要なサポート依頼や情報提供

本リポジトリでは、上記の機能を実装するためのコードや資料を管理していきます。現時点では基本的な設定ファイルや資料のみですが、今後の開発に伴い随時更新していく予定です。
